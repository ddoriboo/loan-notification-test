# 🎯 개인화 맞춤 알림 서비스

대출 서비스의 알림 발송 히스토리를 분석하여 타겟율을 극대화하는 AI 기반 개인화 알림 서비스입니다.

## 📊 분석 결과 요약

### 핵심 지표
- **총 알림 발송 수**: 1,497건
- **평균 클릭률**: 8.45%
- **분석 기간**: 2024년 1월 ~ 2025년 6월 (18개월)
- **서비스 종류**: 8개

### 🏆 TOP 3 성과 서비스
1. **신용점수조회**: 10.84%
2. **신용대환대출**: 8.92%
3. **주택담보대출비교**: 7.83%

### 💡 핵심 인사이트
- **최고 성과 키워드**: '혜택' (10.17% 클릭률)
- **최적 발송 요일**: 수요일 (8.88% 클릭률)
- **이모지 효과**: 클릭률 4.67%p 감소 (사용 지양)
- **발송 규모**: 소규모가 대규모보다 53% 더 효과적

## 🚀 구축된 서비스 기능

### 1. 데이터 분석 엔진
- 18개월 알림 히스토리 종합 분석
- 서비스별/키워드별/요일별 성과 분석
- 고객 반응 패턴 분석

### 2. 고객 세그먼트 분류
- **고반응 세그먼트**: 클릭률 상위 33%
- **중반응 세그먼트**: 클릭률 중위 33%
- **저반응 세그먼트**: 클릭률 하위 33%

### 3. 개인화 메시지 생성
- 세그먼트별 맞춤 메시지 템플릿
- AI 기반 자동 문구 생성
- 성과 예측 기능

### 4. 캠페인 최적화
- 타겟 규모별 최적 배분
- 발송 시간 최적화
- ROI 예측 및 개선 방안

## 📁 파일 구조

```
📦 notification/
├── 📊 데이터
│   ├── 202507_.csv                    # 원본 알림 히스토리 데이터
│   └── 202507_.xlsx                   # Excel 형태 데이터
├── 🔍 분석 엔진
│   ├── simple_analyzer.py             # 핵심 분석 엔진
│   ├── notification_analyzer.py       # 고급 분석 (pandas 기반)
│   └── visualization_generator.py     # 시각화 생성기
├── 🎯 서비스
│   ├── personalized_notification_service.py  # 메인 서비스 (CLI)
│   └── notification_service.py        # 웹 서비스 (Flask)
├── 📄 리포트
│   ├── notification_analysis_report_[timestamp].html  # 시각적 HTML 리포트
│   ├── notification_analysis_data_[timestamp].json    # JSON 데이터
│   ├── service_analysis_detail_[timestamp].csv        # CSV 상세 분석
│   └── analysis_summary_[timestamp].txt              # 텍스트 요약
└── 📖 README.md                       # 이 파일
```

## 🛠️ 사용 방법

### 1. 메인 서비스 실행
```bash
python3 personalized_notification_service.py
```

### 2. 데이터 분석만 실행
```bash
python3 simple_analyzer.py
```

### 3. 시각화 리포트 생성
```bash
python3 visualization_generator.py
```

### 4. 웹 서비스 실행 (Flask 설치 필요)
```bash
python3 notification_service.py
```
